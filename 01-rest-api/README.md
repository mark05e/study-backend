# 01 — REST API

## Overview

Build a simple REST API: single request/response, no state. Every language has a story; this is the best first milestone.

**Why this order:** Single request/response, no state; every language has a story. Best first milestone.

**Minimal spec:** GET/POST `/users`, JSON body/response, one in-memory store.

---

## Per-language notes

| Language   | Notes |
| ---------- | ----- |
| **Python** | FastAPI or Flask. |
| **JavaScript** | Express or Fastify (Node). |
| **Go** | net/http or Gin. |
| **Java** | Spring Web or JAX-RS. |
| **Rust** | Axum or Actix. |
| **Zig** | std.http or zig-http. |
| **C++** | Drogon or cpp-httplib. |
| **C** | libcurl (client) or socket + parser (server). |
